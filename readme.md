# UI SQL vs NoSQL

Una interfaz visual interactiva que explica las diferencias, historia y conceptos clave para entender las bases de datos SQL y NoSQL. Todo en una sola página.

## Qué hay en la web

Está armada en 5 secciones principales que cambian con pestañas:
1. **Contexto Histórico**: Edgar F. Codd, System R y el verdadero nacimiento del modelo relacional.
2. **Laboratorio de Transformación**: Mapeos visuales de cómo una clásica fila SQL se convierte en un objeto NoSQL.
3. **Teoría (pero simple)**: ACID, BASE y el famoso Teorema CAP al grano.
4. **Escalamiento Profundo**: Diferencias entre escalar hacia arriba (vertical) y hacia afuera (horizontal).
5. **Datos Freak 💎**: Curiosidades nerd súper específicas del mundo de las DBs.

## De qué está hecho

- **HTML5** común y corriente.
- **Tailwind CSS** (vía CDN para no complicarnos con dependencias o NPM).
- **Vanilla JS** para manejar la interactividad de las pestañas y las transiciones.
- Fuentes de Google: *Plus Jakarta Sans* y *Fira Code* para los bloques de datos.

## Cómo correrlo

Como no tiene backend ni requiere Node o compiladores, la vida es muy fácil:
1. Bájate los archivos o clona esto.
2. Dale doble clic al archivo SQLvsNoSQL.html para abrirlo en tu navegador favorito.
3. Listo, eso es todo.
